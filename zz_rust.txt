Rust 웹 프레임워크 Actix Web의 App::new()에서 사용하는 주요 메서드는 각각 아래와 같은 역할과 사용 이유가 있다
    1. .wrap : 미들웨어를 어플리케이션에 적용할 때 사용 / 요청 또는 응답 전후에 수행할 공통 작업(로깅, 인증, CORS 등)을 묶어서 처리한다 / 모든 라우트에 영향을 끼친다
    2. .route : 특정 경로 및 HTTP 메서드에 핸들러(함수)를 직접 연결한다 / 라우팅 매크로를 사용하지 않는 수동 매크로 등록 방식이다
    3. .service : 라우트, 리소스, 스코프(범위) 등을 통째로 등록한다 / 여러 경로 또는 복잡한 라우팅 로직, 그룹화된 리소스(스코프/모듈)를 한 번에 추가할 때 사용한다
    4. .app_data : 어플리케이션 전역에서 공유하는 데이터(상태, 설정 등)을 저장한다 / 여러 요청/라우터에서 안전하게 공유해야 하는 상태, 설정값, DB 커넥션 풀 등 등록한다
